Roadmap Prioritas

PRIORITAS 1 — Admin Approval System (Kursus)

Instruktur sudah bisa mendaftar dan membuat kursus, tapi statusnya masih draft. Perlu UI di Admin Dashboard untuk review & publish kursus (mengubah isPublished = true).

PRIORITAS 2 — Sistem Pembayaran (Midtrans)

Integrasikan Midtrans Snap UI untuk pendaftaran kursus premium. Siapkan webhook handler POST /api/payment/midtrans/notification.

PRIORITAS 3 — Video Player (YouTube/Cloudinary)

Isi videoUrl pada tabel Lesson dengan URL iframe nyata. (Opsional) Build fitur upload video dari dashboard instructor ke Cloudinary.

PRIORITAS 4 — Polish UI & Deploy

Navbar active state, perbaiki Todo list dashboard student agar persisten ke database, dan bersiap deploy ke Vercel.
